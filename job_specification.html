Information about Jobs

Junior Software Developer

As a Junior Software Developer, you're going to be responsible for developing and maintaining web applications. Your responsibilities include creating and fixing code, working with the team, and taking part in code reviews.

You should understand at least one programming language, such as JavaScript, Python, or Java, as well as fundamental web technologies like HTML and CSS.

Web Designer

You will create visually appealing and user-friendly web pages, work with developers to implement designs, and ensure consistency throughout all web sites. Experience in design tools such as Adobe XD, Figma, or Sketch, as well as basic HTML and CSS understanding, is necessary.

A degree in web design, graphic design, or a similar discipline, as well as an impressive collection of your work, is required.

Previous internship or project experience is beneficial. Strong imagination, attention to detail, and effective communication abilities are essential.




Web Developer

You will be responsible for designing, coding, and modifying websites, from layout to functionality, in accordance with client demands. HTML, CSS, and JavaScript proficiency are necessary, as is knowledge with frameworks like as React or Angular.

A bachelor's degree in computer science or a related discipline is required, as is excellent problem-solving ability and attention to detail.

Previous expertise with responsive design is preferred, as is knowledge with version control systems such as Git. Good communication and teamwork abilities are also essential.
